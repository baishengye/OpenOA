## 1. 离线 metro bundle 打包（OTA 升级包的核心产物）

> 「离线 metro bundle」= 不连 Metro dev server，把 JS 全量打成一个静态 bundle 文件。`--dev false` 出生产包。**都在 `apps/oa` 跑。**

```bash
cd /Volumes/MacExtend/Code/ReactNative/OpenDingDing/apps/oa

# iOS（产出 main.jsbundle + assets 到 /tmp/x-ios）
rm -rf /tmp/x-ios && mkdir -p /tmp/x-ios
npx react-native bundle --platform ios --dev false --entry-file index.js \
  --bundle-output /tmp/x-ios/main.jsbundle --assets-dest /tmp/x-ios

# Android（bundle 名必须 index.android.bundle）
rm -rf /tmp/x-and && mkdir -p /tmp/x-and
npx react-native bundle --platform android --dev false --entry-file index.js \
  --bundle-output /tmp/x-and/index.android.bundle --assets-dest /tmp/x-and

# 鸿蒙（专用命令，产出固定在 rawfile/bundle.harmony.js）
npx react-native bundle-harmony --dev false --config metro.config.harmony.js
# → apps/oa/harmony/entry/src/main/resources/rawfile/bundle.harmony.js
```

验证 bundle 内容（确认改的标记打进去了）：
```bash
grep -c "OA Demo v3" /tmp/x-ios/main.jsbundle          # ≥1 = 命中
grep -c "OA Demo v3" /tmp/x-and/index.android.bundle
grep -c "OA Demo v3" apps/oa/harmony/entry/src/main/resources/rawfile/bundle.harmony.js
```

## 2. release 安装包 build（内置 bundle，给用户装的基线包）

```bash
cd /Volumes/MacExtend/Code/ReactNative/OpenDingDing/apps/oa

# iOS（模拟器 .app；run-ios release 不连 Metro，bundle 在 build phase 自动打进 .app）
BUNDLE_GEMFILE="$PWD/Gemfile" BUNDLE_PATH="$PWD/vendor/bundle" RCT_NEW_ARCH_ENABLED=1 \
  npx react-native run-ios --mode Release --simulator "iPhone 17"
# .app 路径：~/Library/Developer/Xcode/DerivedData/OpenDingDing-*/Build/Products/Release-iphonesimulator/OpenDingDing.app
# 真机 .ipa 需 Xcode Archive（签名只能在 Xcode 选 Team）

# Android apk（--no-packager 不启 Metro）
JAVA_HOME="/Applications/Android Studio.app/Contents/jbr/Contents/Home" \
  npx react-native run-android --mode release --no-packager --deviceId emulator-5554
# apk 路径：apps/oa/android/app/build/outputs/apk/release/app-release.apk

# 鸿蒙 hap：命令行无 hvigorw → 用 DevEco Studio GUI
#   Build → Build Hap(s)/APP(s) → Build Hap(s)，或直接 Run 装机
#   hap 产出：apps/oa/harmony/entry/build/default/outputs/default/entry-default-signed.hap
```

> ⚠️ Android：重 build apk 会改 build time，导致已下载的 OTA 失效（CodePush「binary 优先」）。基线 apk 定版后别再重 build；要重测先 `adb pm clear`。见 [踩坑速查 #11c](踩坑速查.md)。

## 4. REST API push 到 server（cli release-react 不可用，统一用这个）

```bash
BASE=http://172.16.80.101:3000
TOKEN=$(curl -s -X POST -H "Content-Type: application/json" \
  -d '{"account":"admin","password":"123456"}' "$BASE/auth/login" \
  | sed -E 's/.*"tokens":"([^"]+)".*/\1/')

# 复用函数：cprelease <appName> <Staging|Production> <zip路径> <description>
cprelease() {
  curl -s -X POST -H "Authorization: Bearer $TOKEN" \
    -F "package=@$3;type=application/zip" \
    -F "packageInfo={\"appVersion\":\"1.0.0\",\"isMandatory\":false,\"description\":\"$4\"}" \
    "$BASE/apps/$1/deployments/$2/release"   # 成功返回 {"msg":"succeed"}
}

cprelease OpenDingDing-ios     Staging /tmp/ios.zip "ios v3"
cprelease OpenDingDing-android Staging /tmp/and.zip "android v3"
cprelease OpenDingDing-harmony Staging /tmp/hm.zip  "harmony v3"
```

**坑**：① `package=@x.zip` 后**必须** `;type=application/zip`（否则被当 octet-stream 拒）；② `description` **不能带 emoji**（mysql utf8mb3）；③ `package_hash` 由 server 自动算，不用传；④ `isMandatory:true` = 强制更新（立即重启）。

## 6. 设备端触发升级 + 验证（adb）

> installMode 已统一为 **`ON_NEXT_RESTART`（冷启动生效）**——RN 0.82 新架构下 `ON_NEXT_RESUME` 切后台不可靠。

```bash
ADB=/Volumes/MacExtend/Envirment/Android/sdk/platform-tools/adb
EMU=emulator-5554

# 重置 OTA 状态（重 build apk 后必做，否则旧 OTA 失效）
"$ADB" -s "$EMU" shell pm clear com.opendingding

# 启动 app（之后在「热更新」tab 手点：检查更新 → 同步更新）
"$ADB" -s "$EMU" shell am start -n com.opendingding/.MainActivity

# 彻底冷启动（应用下载的 OTA）
"$ADB" -s "$EMU" shell am force-stop com.opendingding
"$ADB" -s "$EMU" shell am start -n com.opendingding/.MainActivity

# 截图（避开模拟器多屏 warning 污染：存 sdcard 再 pull）
"$ADB" -s "$EMU" shell screencap -p /sdcard/s.png && "$ADB" -s "$EMU" pull /sdcard/s.png /tmp/s.png

# 排障：看 server 收到的 client 请求（package_hash / label / app_version）
cd /Volumes/MacExtend/Code/ReactNative/Server/shm-open-server
docker-compose logs --tail=20 server | grep "<该端key前6位>"
```

iOS 模拟器同理：`xcrun simctl terminate booted org.reactjs.native.example.OpenDingDing` + `launch`，截图 `xcrun simctl io booted screenshot /tmp/s.png`。
